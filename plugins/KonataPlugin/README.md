


## 介绍

> 本项目为 QQ 基础插件, 可通过依赖本插件开发相关插件,  
> 注意: 依赖本插件开发相关插件, 插件将感染 `GPL-3.0 license`

```json
"UseDemoModel": true
```

> 当开启 `UseDemoModel` 时, 可在机器人加入的群内 `@机器人` , 它会自动回复, 此项功能用于测试
