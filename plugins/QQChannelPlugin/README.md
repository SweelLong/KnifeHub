


## 介绍

> 本项目为 QQ频道 基础插件, 可通过依赖本插件开发相关插件

```json
"UseSandBoxMode": true,
"UseDemoModel": true
```

> 当开启 `UseDemoModel` 时, 可在机器人加入的频道内 `@机器人` , 它会自动回复, 此项功能用于测试
